About Me

My name is Jack Burt. I'm 19 years old and I am in my second year at the University of Surrey studying BSc Computer Science.

I've grown up in Worthing, a coastal town in the South of England. In sixth-form I started experimenting with a few small projects using Unity and a raspberry pi, and I discovered I not only enjoyed it but seemed to be pretty good at it.

I've still got a lot to learn but I am always working on something new and improving my skillset. I'm hoping to release my first indie title within the upcoming years, details of which can be found in the projects section of this website.

Besides Unity, I've done some work using Android Studio on a few basic applications, and I am planning to start learning Swift so that I can venture into iOS development too. My first year of Uni consisted of learning mainly in Java so my strongest language is currently that. I also have some experience with HTML, CSS and a little javascript as demonstrated by this website.

Outside of development I've always enjoyed art and design, in both traditional and digital media. Some of my digital works can be found on my projects page. I will have drawn all assets for my game myself as well.
